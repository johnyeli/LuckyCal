# LuckyCal 2026 - 每日幸运色日历

这是一个实验性项目，旨在生成 2026 年的专业 ICS 日历文件。

## 核心逻辑：五行与中国传统色

本项目不使用随机数，而是采用“干支纪日法”结合“五行色彩学”：

1.  **计算干支**：准确计算 2026 年每一天的天干地支。
2.  **五行定性**：根据干支纳音或正五行确定当日能量属性。
3.  **色彩映射**：从精选的《中国传统色谱》中，匹配与当日五行相生或相符的颜色。

## 项目结构

- `data/colors.json`: 中国传统色谱数据库。
- `src/calendar_engine.py`: 核心算法（干支计算与颜色匹配）。
- `src/generator.py`: ICS 文件生成器。
- `main.py`: 主程序入口。
